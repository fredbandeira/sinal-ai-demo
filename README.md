# Sinal.ai

> Plataforma de inteligência política com IA, da pré-campanha ao quarto ano de mandato.

**Demonstração navegável em alta fidelidade** — 13 telas do MVP interligadas, com identidade visual completa, narrativa coerente e persona fictícia (Marina Aguiar, Dep. Estadual por MG).

## 🔗 Ver a demo ao vivo

👉 **[Abrir demo navegável](./index.html)** *(após ativar GitHub Pages, substitua por URL pública)*

## 📐 O que existe aqui

### 13 telas em alta fidelidade, organizadas em 5 núcleos:

**01 · Operação diária**
- Dashboard principal
- Gerador de Posts
- Editor de Post
- Calendário Editorial
- Banco de Respostas
- Atendimento WhatsApp

**02 · Editorial**
- Jornal Diário *(segunda linguagem visual da marca — papel cremoso)*

**03 · Inteligência**
- Escuta Social
- Detector de Fake News
- Análise de Adversários

**04 · Governança**
- Central de Compliance *(Res. TSE 23.732/2026)*
- Configurações e Equipe

**05 · Onboarding do candidato**
- Passo 3 · Sua voz *(fine-tuning da IA)*

### 4 documentos estratégicos
- `Guia-Mestre-Campanha-Deputado.docx` — guia completo de campanha
- `PRD-v0-Plataforma-IA-Campanha.docx` — Product Requirements Document
- `Identidade-v0-Manual.docx` — manual de identidade visual
- `identidade-v0.html` — artefato interativo da marca

## 🎯 Diferenciais do produto

1. **IA treinada na voz do candidato** — não prompt genérico
2. **Conteúdo a partir do trabalho real** — emendas, agenda, votações (+Brasil, SIOP)
3. **Compliance TSE 2026 como feature de primeira classe** — rotulagem, hash, blackout, logs auditáveis
4. **Duas linguagens visuais** — dashboard escuro (ferramenta) + editorial claro (peças compartilháveis)

## 🧭 Como navegar

1. Abra `index.html` (ou a URL pública do GitHub Pages)
2. Clique em qualquer card para entrar na tela
3. Use o sidebar esquerdo dentro de cada tela para navegar
4. Clique no logo **sinal.ai** no topo para voltar ao índice
5. **Trajeto sugerido (10 min):** Dashboard → Editor de Post → Jornal Diário → Fake News → Compliance

## 🎨 Identidade visual

- **Ink:** `#0A0A0B` (fundo)
- **Signal:** `#C5F24A` (cor-assinatura verde-lima, escolhida por não ter conotação partidária no Brasil)
- **Mist:** `#EDEDE9` (texto)
- **Tipografia:** Instrument Serif (display) + Geist (body) + Geist Mono (dados)
